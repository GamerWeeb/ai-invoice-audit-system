# AI Invoice Audit System

An AI-powered invoice auditing tool that detects billing discrepancies such as:

- GST calculation errors
- Duplicate invoices
- Price anomalies
- Incorrect totals

Built using:
- Python
- Streamlit
- OCR (Tesseract)

## Features
Upload invoice PDFs and automatically detect issues.

## Demo
Upload an invoice and see:
- Parsed invoice data
- Audit report
- Vendor-wise discrepancy charts
